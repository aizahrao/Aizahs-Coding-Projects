      About

      Demonstrates several persistent data structures, in this case, shopping lists.

      You can add items to each list by using the text box and the "add item" button.

      You can delete items from the lists by clicking on them.

      The list items are given ids (gid values) so that you can have duplicates.

      You can save the contents of all lists to local storage by using the "save all" button.

      You can clear the saved grocery lists with the "clear" button. That's mostly for debugging.
